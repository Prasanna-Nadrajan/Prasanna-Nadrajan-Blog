2nd Runner up at Code Sapiens ORU-GIT-IN-KADHAI

Contest AI Achievement

I'm thrilled to share that our team secured third place at the oru-git-in-kadhai hosted by CodeSapiens! This contest challenged us to develop an innovative solution using AI to address real-world problems.

Our project, "AI code reviewer and bug fixer," aimed to streamline the software development process by automating code reviews and bug fixes. We utilized advanced machine learning algorithms to analyze code quality and suggest improvements, significantly reducing the time developers spend on manual reviews.
